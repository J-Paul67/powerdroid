Extra APN record is necessary to get GPRS working with Agilent SIM:

	# sqlite3 /data/data/com.android.providers.telephony/databases/telephony.db
	sqlite> INSERT INTO carriers (_id, name, numeric, mcc, mnc, apn, user,
	  server, password, proxy, port, mmsc, current) VALUES (25, 'TestSIM',
	  '001012', '001', '012', 'internet', '*', '*', '*', '', '', '', 1);
	sqlite> .exit


Note that the MNC value is not really right, but is necessary for getting
it to work in current builds.
